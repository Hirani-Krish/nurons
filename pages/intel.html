STATUS: IN PROGRESS | TAG: // LIVE_TRACKING

Real-Time Driver Tracking System

We're building a live GPS tracking dashboard for a logistics client managing 200+ delivery drivers. The system uses WebSocket connections to stream sub-second location updates to a React operations dashboard. We've solved the critical challenge of maintaining tracking accuracy when the driver's phone browser goes to background — using Service Workers and the Background Geolocation API as a fallback layer.

STACK: FASTAPI · WEBSOCKETS · REACT · GOOGLE MAPS API
READ_TIME: ~6 min · STATUS: Active Development

One Website, Two Native Apps — Separating User Flows

Currently engineering a platform where a single website serves as the central hub, while two separate native mobile apps handle distinct user flows — one for customers and one for operations staff. The website shares authentication and data via a common FastAPI backend, but each native app (React Native) is optimized for its specific use case: ordering and tracking for customers, and dispatch management for operators.

STACK: NEXT.JS · REACT NATIVE · FASTAPI · POSTGRESQL
